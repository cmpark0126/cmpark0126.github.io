## 4. 나눗셈

##### 용어 정리

-   A / B = C, A % B = D
-   Dividend: A
-   Divisor: B
-   Quotient: C
-   Remainder: D

##### 나눗셈 알고리즘과 하드웨어

-   7 / 2 = 3, 7 % 2 = 1
-   0111 / 0010 = 0011, 0111 % 0010 = 0001
-   Dividend: 7, Divisor:  2, Quotient: 3, Remainder: 1
-   아래 표에서 순차적으로 보여주고 있다.
